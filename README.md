# Random Raffel Contract

## About

The code that you are going to use is for creating a lottery using a smart contract.

## How does it work?

1. Someone pays for a ticket
   1. The ticket fees will be given to the winner of the contest
2. After a given time the lottery declares a winner
   1. Which will be done by code
3. The process is using chainlink VRF and chainlink automation
   1. Chainlink VRF.. aka Randomness
   2. Chainlink automation.. for Time Based Triggers
